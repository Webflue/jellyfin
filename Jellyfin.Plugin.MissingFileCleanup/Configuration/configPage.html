Missing File Cleanup

This plugin runs as a scheduled task. It walks every selected item type in your library, checks whether the file still exists on disk, and (when enabled) removes the corresponding library entry. The file on disk is never touched.

Dry-run mode is ON. Enable “Allow deletion” below to let the task actually remove entries.
Safety
When unchecked, the task only logs what it would remove.
Abort the run if more than this percentage of scanned items are flagged as missing. Protects against an offline mount wiping your library. Set to 0 to disable.
